--- Step to Install ---
1. Download the code.
2. Run "npm install" to install all node modules.
3. Run "npm run start" to execute the code.

--- Step to Build ---
1. Download the code.
2. Run "npm install" to install all node modules.
3. Run "npm run build" to execute the code.
4. Build folder will be created. 
5. Run the index.html.
